Atexant stands for "Academic TEXt ANalizing Tool" developed in
Computer science department of Tomsk state university and purposed for
research tasks in area of Data Mining, especially designed for text
processing in wikimedia formats.

You can download Atexant sources from the main project git repository
located at: git://github.com/Atexant/atexant.git

Requirements

You need the Java run-time environment 1.6 or higher, Ant tool , the
Wikipedia articles dump and some additional utilities to be able to
compile and run this project. The main English Wikipedia articles dump
can be downloaded from:
http://download.wikimedia.org/enwiki/latest/enwiki-latest-pages-articles.xml.bz2

Additional utilities include Stanford CoreNLP library and JDBC
connector for MySQL.
